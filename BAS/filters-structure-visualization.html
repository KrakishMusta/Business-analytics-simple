Ключевые особенности структуры:

  • Ключ localStorage: 'business-analytics-filters'
  • Структура по полям: Каждое поле аналитики имеет свой ключ
  • Объект поля: Содержит fieldKey и массив filterItems
  • Элементы фильтра: Простые объекты с полем text
  • Автосохранение: Используется useStorage из VueUse
  • Реактивность: Изменения автоматически синхронизируются

Операции с фильтрами:

  • Добавление: handleAddFilterItem() - добавляет новый элемент в filterItems
  • Удаление: handleRemoveFilterItem() - удаляет элемент по id
  • Инициализация: При монтировании создается структура для всех полей
  • Синхронизация: watch() отслеживает изменения в localStorage
